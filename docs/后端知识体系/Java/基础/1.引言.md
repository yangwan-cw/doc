## 1.1 引言
现在编程语言生态十分丰富，比如

| 编程语言   | 应用领域 |
| ---------- | :------: |
| Java       | 后端开发 |
| Python     | 数据分析 |
| C++        | 游戏开发 |
| JavaScript | 前端开发 |
| C#         | 游戏开发 |
| Go         | 后端开发 |

当然，以上表格没有涵盖所有编程语言，只是列举了一些常见的编程语言。应用领域也是将其应用领域进行一个大致的分类，并不是说Java只能做后端开发，Python只能做数据分析。

::: details

:::
