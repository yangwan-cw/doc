# HTML

## 开始我们的第一个 HTML

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <title>Title</title>
  </head>
  <body>
    <h1>hello world</h1>
  </body>
</html>
```



## 注释

```html
<!-- 这是一个 HTML 注释 -->

<!-- 
        下面的代码是一个按钮，点击后将执行 JavaScript
        这个按钮暂时没有添加功能
-->
```

### 注意事项

> [!CAUTION]
> 注释不能嵌套：
> HTML 中不支持注释嵌套，即不能在一个注释中再放另一个注释，如 <!-- <!-- 注释 --> --> 会出错。

> [!NOTE] 重要
>
> - 注释是帮助他人阅读你代码的最好方式
> - 注释不会在浏览器中显示,它仅仅是供开发者参考。
