# HTML 标签

## html 元素

```
<html lang="zh-CN" dir="ltr"><head>
```

html 元素表示 HTML 文件的根,它是所有其他元素的父亲。其他元素必须是此元素的后代

> [!tip]
> 补充额外知识:
>
> - 在 CSS 中 :root 表示的也是 <html></html> 元素.
> - lang 属性的作用: 帮助语音合成工具确认要使用的发音,帮助翻译工具确认翻译的语言

## head 元素

head 元素规定了文档相关的配置信息。常见配置

- 文档标题
- 样式
- Javascript 脚本
- 启用内核版本
