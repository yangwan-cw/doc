# HTML 标签

## html 元素

```
<html lang="zh-CN" dir="ltr"><head>
```

html 元素表示 HTML 文件的根,它是所有其他元素的父亲。其他元素必须是此元素的后代

> [!tip]
> 补充额外知识:
>
> - 在 CSS 中 :root 表示的也是`<html></html>` 元素.
> - lang 属性的作用: 帮助语音合成工具确认要使用的发音,帮助翻译工具确认翻译的语言

## head 元素

head 元素规定了文档相关的配置信息。常见配置

- 文档标题
- 样式
- Javascript 脚本
- 启用内核版本

## body 元素

body 元素里面的元素将是你在网页上显示的内容,我们在编写元素的时候一般写在 body 下

## h1-h6,p 元素

### h 元素使用

```
<h1>标题1</h1>
<h2>标题2</h2>
<h3>标题3</h3>
<h4>标题4</h4>
<h5>标题5</h5>
<h6>标题6</h6>
```

### h 元素

#### h 元素作用

在 HTML 中，h1 到 h6 标签用于定义标题，分别代表从最高级别（h1）到最低级别（h6）的标题。它们通常用于结构化文档，帮助搜索引擎和用户快速识别重要内容。这些标题标签通常用于文章、页面或内容的不同部分，提供层级关系和可视化分隔。

##### 问题思考：h 元素如何区分 h1-h6 字体大小

h1 到 h6 标签的区别主要体现在字体的大小、粗细以及在页面中所扮演的角色。具体来说：

`默认字体大小`：

浏览器默认为 h1 到 h6 元素设定了不同的字体大小，h1 最大，h6 最小。例如：

h1：通常为页面中最重要的标题，默认字体大小较大，可能为 2em 或 32px。

h2 到 h5：随着级别递减，字体大小也逐渐减小。

h6：最小的标题，通常为最小的默认字体大小。

这些默认的字体大小是由浏览器的样式表设置的，但可以通过 CSS 来进行自定义和调整。

> [!tip]
> 总结: 浏览器默认为 h1-h6 设置了一些默认样式

## img,a,iframe 元素

### img 使用

```
<img src="image-128.png" srcset="image-256.png 2x" alt="error"/>
```

### img 作用

img 用于在页面上插入一份图片

- src 图片资源路径
- srcset 用于设置不同屏幕密度下，img 会自动加载不同的图片
- alt 图片资源获取或加载不成功时,所显示的文本

#### 特殊知识点

- img 会请求资源,并将 img 占用的位置嵌入
- img 是单标签

#### img 的 srcset 属性的作⽤？

响应式页面中经常用到根据屏幕密度设置不同的图片。这时就用到了 img 标签的 srcset 属性。srcset 属性用于设置不同屏幕密度下，img 会自动加载不同的图片。用法如下：

```
<img src="image-128.png" srcset="image-256.png 2x" />
```

使用上面的代码，就能实现在屏幕密度为 1x 的情况下加载 image-128.png, 屏幕密度为 2x 时加载 image-256.png。

按照上面的实现，不同的屏幕密度都要设置图片地址，目前的屏幕密度有 1x,2x,3x,4x 四种，如果每一个图片都设置 4 张图片，加载就会很慢。所以就有了新的 srcset 标准。代码如下：

```
<img src="image-128.png"
     srcset="image-128.png 128w, image-256.png 256w, image-512.png 512w"
     sizes="(max-width: 360px) 340px, 128px" />
```

其中 srcset 指定图片的地址和对应的图片质量。sizes 用来设置图片的尺寸零界点。对于 srcset 中的 w 单位，可以理解成图片质量。如果可视区域小于这个质量的值，就可以使用。浏览器会自动选择一个最小的可用图片。

```
sizes="[media query] [length], [media query] [length] ... "
```

sizes 就是指默认显示 128px, 如果视区宽度大于 360px, 则显示 340px。

## a 元素

a 元素可以让用户点击，快速跳转到页面的不同位置,或者跳转到其他网页

#### 案例

想象一下，你在读一本书，书的内容有很多章节。每个章节的开始都会有一个标题。如果你想从书的目录直接跳到某个章节，你可以点击那个章节的标题，然后页面会自动滚动到那个章节的位置，这就是锚点的作用。

#### 使用

##### 页面内进行跳转

```
<!-- 锚点位置 -->
<h2 id="section1">这是第一部分</h2>
<p>这里是第一部分的内容...</p>

<!-- 链接到锚点 -->
<a href="#section1">跳到第一部分</a>
```

##### 跳转到其他页面

```
<a href="https://www.example.com#section2">跳到 Example 网站的第二部分</a>

```

#### 其他属性

> [!tip]
> target: 指定 a 标题在何处显示资源
>
> - self: 默认值, 当前窗口打开 URL
> - \_blank: 新的窗口打开 URL

#### 额外补充

a 元素不仅仅只做跳转,这与 href 属性相关,href 内是压缩包地址,那么 a 元素的动作是下载压缩包。herf 内是邮箱地址,那么 a 元素动作是发送邮件

#### iframe 元素

iframe 用于在 HTML 文档中嵌入一个 HTML 文档

##### 如何使用

```
<iframe src="http://www.baidu.com" width="" height="" frameborder="" />
```

- src: 资源地址
- width: 文档宽度
- height: 文档高度
- frameborder: 是否显示边框

##### 如何让自己的网址禁用别人 iframe

- 使用 X-Frame-Options 响应头
- 使用 Content-Security-Policy 头

## div,span 元素

### `div` 元素

- **作用**：`<div>` 是一个块级元素，元素在网页结构中,独占一行,用于将页面内容进行分组。它没有任何特定的语义，仅仅是一个容器。通常与 CSS 一起使用来控制布局和样式。
- **使用场景**：`<div>` 常用于创建页面的结构，比如页面的头部、内容区域、侧边栏和页脚等。
- **示例**：
  ```html
  <div class="container">
    <header>
      <h1>网站标题</h1>
    </header>
    <main>
      <p>这是主页的内容。</p>
    </main>
    <footer>
      <p>&copy; 2025 网站名称</p>
    </footer>
  </div>
  ```

### span 元素

作用：`<span>` 是一个行内元素,多个 span 元素共同占一行,用于包裹文档中的小部分内容。它不会对布局产生任何影响，主要用于应用样式或操作文本。

使用场景：`<span>` 适用于局部文本的样式应用，比如改变字体颜色、加粗或加斜等。

示例：

```
<p>我喜欢 <span style="color: red;">红色</span> 和 <span style="font-weight: bold;">粗体字</span>。
```

## 不常用的 HTML 元素

1. `<abbr>`

   - 用途：标记缩略词，通常带有 `title` 属性显示全称。
   - 示例：`<abbr title="World Health Organization">WHO</abbr>`

2. `<dfn>`

   - 用途：定义术语，通常首次出现某个术语时使用。
   - 示例：`<dfn>HTML</dfn> 是一种标记语言。`

3. `<mark>`

   - 用途：高亮文本（通常用于搜索结果）。
   - 示例：`这是一段被 <mark>标记</mark> 的文字。`

4. `<details>` 和`<summary>`

   - 用途：创建可展开/折叠的内容块。
   - 示例：
```
<details>
       <summary>点击查看详情</summary>
       <p>这是详细信息内容。</p>
     </details>
```

5. ``<s>``

   - 用途：表示不再准确或已废弃的内容（删除线效果）。
   - 示例：`<s>原价 $100</s> 现价 $70`

6. ``<kbd>``
   - 用途：表示键盘输入。
   - 示例：`按下 <kbd>Ctrl</kbd> + <kbd>S</kbd> 保存`

## HTML 全局属性简述

- **id**  
  唯一标识元素，用于 CSS 或 JavaScript 定位。

- **class**  
  分配一个或多个类名，可供 CSS 或 JavaScript 使用。

- **style**  
  行内样式，用于为元素直接设置 CSS 属性。

- **title**  
  鼠标悬停时显示的提示文本，增强可读性或辅助信息。
