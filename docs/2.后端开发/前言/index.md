## 为什么学习 Java 
是人是狗现在都知道 Java 如今非常的 **内卷**，内卷属实没边了，但是 Java 我个人还是推荐
学习的，有以下几个点，提供给各位参考参考
- 卷的人多，但是不够精
- web 生态真的非常成熟
- 上手相对 C++ 偏简单点

上方的点，属于自己经历中所汇总出来的，内卷哪个行业都内卷。但是只想告诉你一点的是：**如果选择
编程那么你的路永远不可能只是 Java.**

## Java 的应用场景
针对这个点，你一定要了解一个语言的应用场景，真的非常非常重要，Java 的应用场景非常的多，但是
主流的场景主要是以下几点：

- 安卓
- web
- 大数据


Java 其实大多数场景面对的是这三个，当然其他方向也是可以走的，但是这三个场景是 Java 主流的场景.
