# <i class="fa-brands fa-python"></i> python 前言

## 为什么学习 python 

1. 学起来相对其他语言会更加简单
2. 哪里都能用到
3. 有问题？社区帮你解决
4. 无论Windows, macOS, 还是Linux都能跑
5. 提供超多的库和框架
6. 修改和扩展都很方便

上面这些听起来是不是很熟悉？没错，很多编程语言都能提供这些优点。我对上方的优点进行提炼，其中学习 python 最主要的驱动力是: 

1. 上手快，学了就能用。
2. Python 就像瑞士军刀，什么都能做，而且做得快。

::: warning 
当然，学习 Python 的时候也不能只看到好的一面，它也有不足之处，比如：

1. 运行速度可能不如一些编译语言快
2. 如果要深入底层，可能会比较困难
:::
