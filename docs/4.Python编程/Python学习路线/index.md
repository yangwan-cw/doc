# 学习前言
在学习一门语言之前笔者非常有必要去了解以下知识，才能更好的学习这门语言。
1. [编程概念](./编程概念)
2. [开发概念](./开发概念)
3. [408](./408)

通过点击 **以上链接** 可以查看详细内容。
:::tip
为什么作者要推荐花费时间去学习这些知识，而不是直接学习Python，原因如下：
1. 编程概念是学习任何编程语言的开门砖，通过开门砖理解编程概念，你的大脑会形成一个基本地基。
2. 开发术语是学习开发中常见的术语，通过开发术语的学习，可以对一些术语上形成一个初步的了解。
3. 408是计算机专业的必修课，通过408的学习，可以让你在计算机专业知识上更加扎实。
:::
